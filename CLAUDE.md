# Mickey Maroulis — Portfolio Website

## Overview
A dark, minimalist portfolio site for Mickey Maroulis, a director and cinematographer. Built as a static site (HTML/CSS/JS) for GitHub Pages deployment at `mickeymaroulis.github.io`.

## Design Direction
- **Dark/black background** with simple white text
- **Monospace typography** (e.g., JetBrains Mono, Space Mono, or similar)
- **Minimalist** — let the video work speak for itself
- **Non-traditional UI touches** — custom scroll bar, subtle hover effects, unconventional but intuitive navigation
- **Visually appealing** — clean spacing, cinematic feel
- **Highly intuitive navigation** — easy to move between sections
- **YouTube embeds** for all video content (no thumbnails linking out)

## Tech Stack
- Plain HTML, CSS, JavaScript (no frameworks)
- No build step — GitHub Pages serves static files from `main` branch root
- `index.html` as entry point

## Site Structure

### Pages / Sections
1. **Home** — Name, title ("director // cinematographer"), entry point to work
2. **Narrative** — Film projects with YouTube embeds
3. **Short-form** — Travel films, magazine shoots, music videos with YouTube/Instagram embeds
4. **Video Art** — Experimental pieces with YouTube embeds
5. **Info** — Contact and social links

### Navigation
- Persistent, minimal nav
- Smooth transitions between sections
- Custom scrollbar or scroll indicator
